Java概述与环境搭建

1.程序

程序是为了模拟现实世界，解决现实问题而使用计算机编写的一系列有序的指令集合。

2.Java平台

（主要记住JDK5.0和JDK8.0即可，JDK8.0后开始收费)

JavaME：针对消费类电子设备，如可视电话、数字机顶盒、汽车导航系统

JavaSE：定位在客户端，主要用于桌面应用软件的编程

JavaEE：用于开发装配以及部署企业级应用程序，主要包含Servlet、JSP、Javabean、JDBC、Web等技术

3.Java语言特点

  • 面向对象（贴近人类思维模式，模拟现实世界，解决现实问题）
  • 简单性（自动内存管理机制、不易造成内存溢出，如：自动释放内；简化流程处理、语义清晰）
  • 跨平台（操作系统、服务器等，写一次到处运行）

4.计算机的执行机制

编译执行：执行效率高，不可跨平台（编译器操作系统自带）

解释执行：执行效率低，可跨平台（解释器不自带，如office、WPS）

5.Java的执行机制

先编译再解释：（先安装环境JVM再写.java编译再生成.class再运行）

JVM：java虚拟机

名词解释

  • JVM（Java Virtual Machine）虚拟机：使用软件在不同操作系统中，模拟相同的环境
  • JRE（Java Runtime Environment）运行环境：包含JVM和解释器，完整的Java运行环境
  • JDK（Java Development Kit）开发环境：包含JRE+类库+开发工具包（编译器+调试工具）

6.基础DOS命令

更换盘符：盘名+:；

查看当前目录下的文件及文件夹：dir；

进入文件夹：cd 文件夹的名字；（cd全名change directory 改变路径，tab可以快速选择文件夹）

返回上级目录：cd..;（返回根目录为cd/或cd\）

清空屏幕：cls;

删除文件：del 文件名；

删除文件夹：rd 文件夹名；（rd全名remove directory）

退出：exit;（关闭窗口）
